RSA-tools - Neighbour ORFs manual

Name

    neighbour-orfs

    1997-98 by

Description

    Returns the closest ORFS on left and right side from a given chromosomal coordinate.

Options

    Query: each line is considered as a separate query. The query list can take three alternative formats:

    • ORF identifier: the first word of each line contains an ORF identifier (e.g.: YFL021W).
    • Chromosomal positions:
      • The first word of each line is the chromosome number (in latin or arab notation).
      • The second word is the start position
      • The 3d word is the end position (optional)
      • The 4th word is an identifier (optional)
      Examples of chromosomal positions
        IX 20000 (a single position in the chromosome IX)
        XIII 20000 40000 (a pair of coordinates specifying a segment of chromosome XIII
    • Genome search result: the output of a whole genome search

    Output format
    Each line of the output correspond to one line fro the query.

      upstream-downstream

      1. Q_id query identifier
      2. Q_gene query gene name
      3. Q_chr query chromosome
      4. Q_start query start position
      5. Q_end query end position
      6. U_ORF upstream neighbour ORF identifier
      7. U_gene upetrsam neighbour gene name
      8. U_dist distance between query position and upstream neighbour
      9. D_ORF downstream neighbour ORF identifier
      10. D_gene downstream neighbour gene name
      11. D_dist distance between query position and downstream neighbour

    left-right

    1. Q_id query identifier
    2. Q_gene query gene name
    3. Q_chr query chromosome
    4. Q_start query start position
    5. Q_end query end position
    6. L_ORF left neighbour ORF identifier
    7. L_gene left neighbour gene name
    8. L_dist distance between query position and left neighbour
    9. R_ORF right neighbour ORF identifier
    10. R_gene right neighbour gene name
    11. R_dist distance between query position and right neighbour

Link to database
When this option is active, ORF identifier are automatically linked to the correspondent entry in a yeast genome database (Standford or MIPS).
