Programs

We are grateful to Stefan Kurtz (kurtz@TechFak.Uni-Bielefeld.DE) for allowing us to install on the server his programs mkvtree and vmatch, which we use to purge sequences before pattern discovery.

consensus and patser have been written by Jerry Hertz (hertz@colorado.edu). The web interface is by Jacques van Helden.

The gibbs sampler has been written by Andrew Neuwald (neuwald@cshl.org). The web interface is by Jacques van Helden.
