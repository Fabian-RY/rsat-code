RSA-tools - Web Services

For bioinformaticians, RSAT tools are now also available as Web Services.

Web Services offer a programmatic interface for running the tools from a remote computer, and combining them with other similar services. Using RSAT via Web Services permits to automatize repetitive tasks.

  • The WSDL description of the services, which is the primary programmatic interface for the web clients, can be accessed at: http://rsat.scmbb.ulb.ac.be/rsat/web_services/RSATWS.wsdl

  • Full description of the available Web Services and their options

  • Examples of web clients in Perl, java and Python (as a downloadable archive)

  • Tutorial with some documented examples of implementations.
