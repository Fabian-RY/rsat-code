RSA-tools - Tutorials - genome-scale dna-pattern

We are now considering the case where we know the binding specificity for a given transcription factor, and we are looking for all the genes potentially regulated by this factor. We will use the Nitrogen regulation as an example. Response to nitrogen is mediated by the so-called GATA-box, whose most common motif is GATAAG. An interesting property is that a single GATA-box is insufficient to affect the transcription. All genes effectively controlled by these elements possess at least one group of 2 to 4 closely associated GATA-boxes.

We will use dna-pattern to select all genes having 3 or more occurrences of the GATA box in their 500 bp upstream region.

  1. On the left pannel, click genome-scale dna-pattern (strings).
  2. In the Sequence retrieval options, make sure that the selected organism is Saccharomyces cerevisiae, and that Sequence type is set to upstream.
  3. Specify the sequence limits from -500 to -1.
  4. In the Query pattern(s) box, type
    5. GATAAG
  5. In the Return option, select match counts.
  6. Change the Threshold to 3.
  7. Click GO

After a few seconds you will see a list of the genes with 3 or more occurrences of the GATAAG pattern in their 500 bp upstream sequence. Each ORF identifier appears as a hyperlink, which is connected to the information on this ORF in the yeast genome database.
