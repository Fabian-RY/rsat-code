RSAT - Drosophila melanogaster ORegAnno Motifs

These motifs were generated from transcritpion factor binding sites annotated from the Open Regulatory Annotations Database (ORegAnno).

  • All motifs were generated using Tom Bailey's meme program from annotated TFBS in ORegAnno. Motifs were created only for TF having more than 2 TFBS.
  • For each TF the we created the PSSM, the information content, the logo and several motifs parameters.
