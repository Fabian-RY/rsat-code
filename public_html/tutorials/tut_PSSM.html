Introduction

    Position-specific scoring matrices (PSSM) offer a sensitive way to represent the specificity of trnscription factor/DNA interfaces. PSSM are built on the basis of a set of known binding sites for the factor of interest.

    Knowledge-based PSSM can be obtained from various transcription factor databases (e.g. TRANSFAC).

    PSSM can also be inferred by applying matrix-based pattern discovery in regulatory regions of a set of co-regulated genes (see tutorials on the gibbs sampler and consensus).

Example

    The matrix below was obtained from SCPD, promoter database of the yeast Saccharomyces cerevisiae. It has been built from an alignment of 12 binding sites for the yeast transcription factor Pho4p. 
    A |  3   2   0  12   0   0   0   0   1   3
C |  5   2  12   0  12   0   1   0   2   1
G |  3   7   0   0   0  12   0   7   5   4
T |  1   1   0   0   0   0  11   5   4   4

    Each row represents one residue (A, C, G or T), and each column a position in a set of aligned binding sites. Some positions are perfectly conserved across all known binding sites (the motif CACGT starting at the 3rd position), whereas some other positions present two choices (e.g. G or T at position 8), and other position can contain any letter, but with different frequencies (e..g first and last position).

    When the matrix will used to scan sequences for putative Pho4p binding sites, the more conserved positions will impose stronger constraints. This allows thus a more quantitative evaluation of the binding cappability than string-based representations.

    One weakness of these matrices is that they do not take into account higher order dependencie, i.e. influence between a residue and the successive ones. Thus, it is not possible to specify "either CACGTGGG or CACGTTTT" with a PSSM : if one builds a matrix where G and T are allowed at the 3 last positions, any combination of them will be allowed (e.g. CACGTGTG, CACGTTGT).
