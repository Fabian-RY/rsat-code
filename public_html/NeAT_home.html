Network Analysis Tools

These last years, large scale biological studies in many different fields produced a huge amount of data that are often represented as biological networks. Moreover, the global understanding of large scale protein-protein interaction, gene expression, localization, and regulation studies is a key challenge in modern systems biology.

For this purpose, we developped the Network Analysis Tools (NeAT), as set of tools allowing classical network manipulations.
