RSA-tools - Credits

Data

Most genomes (especially those of prokaryotes) are parsed from the NCBI genome repository.

    ftp://ftp.ncbi.nih.gov/genomes/

Some additional genomes (especially those of metazoan) are imported from the ENSxEMBL database.

    ftp://ftp.ensembl.org/pub/

RSAT logo (August 2014)

The RSAT logo was designed by Mauricio Guzmán (http://www.altamirastudio.com/)

Third-pary Programs

We are highly grateful to Stefan Kurtz (kurtz@zbh.uni-hamburg.de) for allowing us to install on the server his programs mkvtree and vmatch, which we use to purge sequences before pattern discovery. These two programs have been faithful companions of RSAT since 1999.

convert-matrix uses the program seqlogo to return sequence logos. This program is from the WebLogo package developed by Gavin E. Crooks, Gary Hon, John-Marc Chandonia and Steven E. Brenner.
