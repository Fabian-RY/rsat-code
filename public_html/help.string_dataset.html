Description

    Download a subgraph from the STRING database.

Authors

Organism

    Select the organism to which the query genes belong.

Requested genes

    Enter the query genes for which you want to know their neighbours in STRING and the interactions between them.

Thresholds

    Each interaction annotated in STRING presents a significance score (between 0 and 1). You can put an upper or a lower threshold on this score.

Evidence channels

    STRING is an integrative database, i.e., the provided functionnal interaction information come from different sources. You can choose to select interactions coming from different combinations of evidence channel. By default, combined evidence is selected. This evidence channel combines all the others channels and a global significance score is calculated according to them.
    • Textmining : coocurence of the gene names in litterature
    • Experimental interactions
    • Gene coexpression : micro-array experiments
    • Gene co-occurence (phylogenetic profiles)
    • Gene fusion events
