RSA-tools - Help about purge-sequence

Program developed by Stefan Kurtz (kurtz@TechFak.Uni-Bielefeld.DE).
Web interface by Jacques van Helden (jvanheld@ucmb.ulb.ac.be.

Description

Remove large duplicate fragments from a sequence. This perl script is no more than a wrapper to facilitate this specific usage for two programs developed by Stefan Kurtz : mkvtree and vmatch.

When a fragment is duplicated, the first ocurrence if it is conserved and the second deleted. By default, the program also searches similarities between the direct and reverse complement strand.

This web interface is no more than a wrapper around two programs (mkvtree and vmatch) developed by Stefan Kurtz.

Options

    Purge both strands
    Duplicates are also searched between the direct and the reverse complementary strand.

    Minimal match length

    Maximal number of mismatches
