RSA-tools - Tutorials - Pattern discovery

    Starting from the simple knowledge that a set of genes share some regulatory behaviour, one can suppose that some elements are shared by their upstream region, and one would like to detect such elements. We implemented a simple and fast method to extract such elements, based on a detection of over-represented oligonucleotides. This method has been described in detail in J.Mol. Biol. (1998) 281, 827-842.

    We will illustrate this approach with a set of genes for which the regulatory sites is well known : the Nitrogen-responsive genes. The pattern discovery program will of course not receive any indication about these known regulatory motifs. We will only use them a posteriori to compare the discovered modifs and those documented in the literature.

    1. Retrieve upstream sequences from -800 to -1 for the following yeast genes (as you have seen in the tutorial on sequence retrieval) In the menu from the left frame, click on retrieve sequence.
      2. DAL5
GAP1
MEP1
MEP2
MEP3
PUT4
DAL80

    2. After a few seconds, the upstream sequences from the 7 Nitrogen sensitive genes are displayed. At the top of the response, click the button labelled oligonucleotide analysis.

    3. a new form appears. In the title box, type
      4. Nitrogen sensitive genes

    4. Leave all other parameters unchanged and click GO.

    5. The results of the analysis are now displayed in columns, showing respectively

      1. the oligonucleotide sequence

      2. the same followed by its reverse complement

      3. the expected frequency for this specific oligonucleotide

      4. the total number of occurrences observed in the set of upstream regions

      5. the number of occurrences expected at random

      6. the probability to observe as much or more occurrences

      7. the significance of the observed number of occurrences

      Note the particularly high significance index (20.00) for the cttatc|gataag oligonucleotide. This corresponds to the GATA box consensus, which is in fact the regulatory site characteristic for the Nitrogen sensitive genes.

      Notice that the value of 20.00 is the maximum possible significance (actually this value is shown when the the probability is too low to be calculated precisely : we reached the floating point limitation of the computer).

    6. In the top of the result page, click on the button Pattern search. The same form as in the part 1 is displayed, but the Query pattern box is now filled with the result of oligonucleotide analysis. Do not change anything to the form and click GO.

    7. the list of positions of the over-represented oligonucleotides is displayed. On the bottom of the result page, click the Feature map button.

    8. The feature map form appears. Note that the title box is already filled.

    9. Fill

      • the from box with -600,

      • the max box with 0

    10. click GO.

    11. On the feature map, note the high number of CTTATC (=GATAAG) sites.
