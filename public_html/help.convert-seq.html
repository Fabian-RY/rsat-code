RSA-tools - Help about sequence conversion

Description

    Performs conversion between various formats of DNA sequences.

Options

    Input format

    Output format

    Line width
    A carriage return is inserted every # characters. A value of 0 prevents the insertion of carriage returns.

    Add reverse complement
    Each sequence from the input file is followed by its reverse complementary sequence.
