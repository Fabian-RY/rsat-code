Voici attaché un fichier de controle pour la présentation en
formulaire de données séparées par des TAB dans emacs.

NB la première ligne définit le nom du fichier qui contient les
données; la fin du fichier définit les différents champs et comment
ils seront présentés.
Pour mettre en route il faut faire la commande
        Esc-X  forms-find-file
puis donné le nom du fichier data.fctl (pas le nom du fichier
contenant les données)
Le premier formulaire, comme suggéré par emacs est affiché avec C-c C-o
