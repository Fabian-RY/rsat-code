Click on the buttons to display separately the information of each cluster.
  • Logo tree and alignment. This results shows the hierarchical tree with all the motifs of the corresponding cluster and the logo alignment in F (left logo) and R (rigth logo) orientation.
  • Branch consensus. Each branch show the branch-consensus which summarizes all the descendant motifs. Click on it to see the corresponding logo.
  • Branch-motifs. On the logo tree, is displayed the the branch-consensus which summarizes all the descendant motif of each branch. This table shows more detailed information: the logo in both orientation and a file in TRANSFAC format corresponding to the branch-motif.
