RSA-tools - Tutorials - retrieve-seq

In the left pannel, under the title Sequence retrieval, click on the tool "Retrieve sequence". A form appears, allowing you to select the parameters of sequence retrieval.

Retrieving upstream sequences for a selected set of genes

    We will first retrieve upstream sequences for a set of 5 yeast genes involved in yeast phosphate metabolism.

    1. In the pop-up menu "Organism", select Saccharomyces cerevisiae

    2. In the text area under "Genes", type the names of your genes
      3. YBR093C
PHO8
pho11
PHO81
Pho84
      Warning: gene names must be separated by carriage returns, because only the first word of each line is considered as a query.

      Notice that gene names are case-insensitive.

      Genes can be specified either by the systematic ORF identifier or by a common name. Synonyms are supported.

    3. Leave all other options unchanged and click on the button GO

    After a few seconds, the upstream sequence of the selected genes will be displayed.

    At the bottom of the sequences, you can see a series of buttons, which will allow you to send the retrieved sequences to the next task (see other tutorials).

Retrieving all downstream sequencs

    As an illustration fo the retrieval of all sequences, we will retrieve the complete set of downstream sequences for the yeast Saccharmoyces cerevisiae

    1. Come back to the previous form by clicking retrieve sequence in the left pannel.

    2. Make sure that the selected organism is Saccharomyces cerevisiae

    3. Besides the label Genes, select "all".

    4. Do not enter anything in the "Gene" box (it would be ignored anyway).

    5. Select "downstream" as Sequence type.

    6. In the From box, type "1".

    7. In the to box, type "200".

    8. Select "ORF id + Gene name" in the pop-up menu Sequence label

    9. Click GO

Reference point

    The reference point (coordinate 0) depends on the type of sequences to be retrieved :

    • For upstream sequences, the reference point is the ORF start, i.e. the first nucleotide of the start codon. Negative coordinates are used to indicate sequences located upstream the start codon, i.e. in the 5'end flank of the gene (e.g. from -1 to -800 for the yeast).
    • On the contrary, when selecting downstream sequences, the reference point (position 0) is the ORF end, i.e. the last nucleotide of the stop codon. Positive coordinates are used to represent sequences located downstream the stop codon, i.e. the 3'end flank of the gene.

Positive/negative coordinates

    Notice that selecting upstream sequencs with positive coordinates would amont to pick coding sequences on the 5' side of the gene. For example, one can extract all start codons for a given organism by retrieving all upstream sequences from position 0 to position +2. Try it.

    Recprocally, for downstream sequences, negative coordinates could be specified to retrieve coding sequences on the 3'en of the gene. One could thus retrieve all downstream sequences from -2 to 0 to extract all the stop codons found in a selected organism.
