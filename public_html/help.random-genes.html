RSA-tools - Help about random-genes

Description

    Random-genes performs a random selection among the genes of a selected organism. The selection can be performed with or without replacement.

    This program is useful for estimating the rate of false positive for pattern discovery programs.

Options

    Number of genes
    Number of genes to be returned.

    Number of groups
    The prorgam can generate several group of rndom genes. This is very convenient to simulate the results of a clustering.

    Replacement
    When this option is activated, a gene can appear several times in the list.

Output format

    The output is a two-column text. The first column gives the gene identifier, the second column the group identifier (useful when several groups are exported).
