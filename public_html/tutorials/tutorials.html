RSA-tools - Tutorials

The aim of this document is to present the Regulatory Sequence Analysis Tools (RSAT). The most convenient way to follow the tutorial is to display the current page in a separate window, and to use the tools with the current one.

(click here for opening a new window with the tutorial pages)

RSAT home page displays two frames. The left frame contains a menu, presenting the tools available. Each time you click on a tool in the left frame, the right frame shows you the form page for the corresponding tool. The tools are interconnected, allowing you to send automatically the result of one request as input for the next request (piping). The links between tools are illustrated in the flow chart below.

We will analyze some practical examples to get familiar with the different tools, and the way they are interconnected. The tutorial contains different parts, illustrating the typical situations one can face when analysing regulatory sequences :

  1. You know the regulatory motif (e.g. the consensus for a transcriptional factor), and you know the genes: you look for the matching positions within the upstream regions from your genes (pattern matching);
  2. You know the regulatory motif, you ignore the genes : you want to scan the genome to detect genes having this motif in their regulatory regions, which are potential target genes for your transcription factor (genome-scale pattern matching);
  3. You know the genes, you ignore the regulatory motif : you look for elements shared by a set of functionally related upstream sequences, which could reveal unknown regulatory sites (pattern discovery).

Tutorials

  1. Sequence retrieval
  2. Representations of regulatory signals
    • String-based representations
    • Position-specific scoring matrices (PSSM)
  3. Pattern matching
    • dna-pattern string-based pattern matching
    • patser matrix-based pattern matching
  4. Pattern discovery
    • String-based pattern discovery
      • oligo-analysis detection of over-represented words
      • dyad-analysis detection of over-represented pairs of words
      • position-analysis detection of words having a positional bias in sequences aligned on some reference position (e.g. the start codon).
    • Matrix-based pattern discovery
      • gibbs motif sampler (program developed by A.Neuwald))
      • consensus (program developed by Jerry Hertz))
    • Applications of pattern discovery
      • Microarrays prediction of regulatory motifs from clusters of co-expressed genes.
  5. Genome-scale pattern matching
    • genome-scale dna-pattern (string-based)
    • genome-scale patser (matrix-based)
  6. Random models
    • Selecting random genes
    • Generating random sequences
