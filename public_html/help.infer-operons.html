RSA-tools - infer-operons manual

DESCRIPTION

    Given a list of input genes, infer the operon to which each of these genes belong.

    The inference is based on a very simplistic distance-based method, inspired from the Salgado & Moreno-Hagelsieb method (Proc Natl Acad Sci U S A. 2000;97:6652-7). The Salgado & Moreno-Hagelsieb method classifies intergenic distances as TUB (transcription unit border) or WO (within operon), and infers operons by iteratively collecting genes until a TUB is found. In the original method, the TUB or WO assignation relies on a log-likelihood score calculated from a training set.

    The difference is that we do not use the log-likelihood (which presents risks of over-fitting), but a simple threshold on the intergenic distance. Thus, we infer that the region upstream of a gene is TUB if its size is larger than a given distance threshold, and WO otherwise. Our validations (Rekin&rsquos Janky and Jacques van Helden, unpublished results) show that a simple threshold on distance raises a similar accuracy as the log-likelihood score (Acc ~ 78% for a threshold t=55).

ALGORITHM (DISTANCE THRESHOLD)

The algorithm is based on three simple rules, depending on the relative orientation of the adjacent genes.

Rule 1: divergently transcribed genes

If the gene found upstream of a query gene is transcribed in the opposite direction, then the intergenic region is considered as a TUB, and the two flanking genes are labelled as operon leaders. This prediction is reliable (as far as genome annotation is correct), since operons only contain genes on the same strand.

Rule 2: convergently transcribed genes

If the gene found downstream of a query gene is transcribed in the opposite direction, then the intergenic region is considered as a TUB, and the two flanking genes are labelled as operon trailers. This prediction is reliable (as far as genome annotation is correct), since operons only contain genes on the same strand.

Rule 3: tandem genes (adjacent genes on the same strand)

If two adjacent genes are on the same strand, then a distance threshold (option -dist) is applied to decide whether they belong to the same operon (dist <= threshold) or not (dist > threshold). If they are predicted to be in distinct operon, the upstream gene is labelled as operon trailer, and the downstream gene as leader of the next operon.

Genes

Each row of the input specifies one query gene (similarly to rerieve-seq). The first word of a gene is the query, the rest of the row is ignored.

OUTPUT FIELDS

leader

Predicted operon leader.

trailer

Predicted operon trailer.

operon

Full composition of the operon. The names of member genes are separated by a semi-column ``;''.

q_info

Detailed info on the query gene(s).

up_info

Detailed info on the upstream gene.

down_info

Detailed info on the downstream gene.
